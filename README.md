## Installation

 1. Get it from GitHub

 ```shell
 $ git clone https://github.com/iTimeTraveler/hexo-theme-hipaper.git themes/hipaper
 ```
 2. Enable

 Modify `theme` setting in `_config.yml` to `hipaper`.
 ```
 # Extensions
 ## Plugins: http://hexo.io/plugins/
 ## Themes: http://hexo.io/themes/
 theme: hipaper
 ```
 3. Update

 ```shell
 $ cd themes/hipaper
 $ git pull
 ```
